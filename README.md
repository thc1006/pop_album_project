# 現代流行專輯：台灣風情

這是使用 Strudel live coding 技術創作的一張概念流行專輯。歌曲融合了流行搖滾、龐克、民謠、合成器流行和舞曲等元素，靈感來自台灣知名樂團美秀集團和告五人。每首作品的程式碼都存於 `src/` 目錄，可直接複製到 [Strudel 編輯器](https://strudel.cc) 播放。

## 如何使用

1. 前往瀏覽器中的 Strudel 編輯器。
2. 從 `src/` 中打開某首歌曲的 `.js` 檔案，複製全部內容貼到 Strudel。
3. 按下 Ctrl+Enter 即可播放音樂。
4. 若想修改歌曲，可在檔案內調整 pattern，或使用 `motifs_library.js` 中提供的動機。
