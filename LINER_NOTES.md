# 詳細曲解 (Liner Notes)

## 1. Intro（序曲）
柔和的 pad 和低沉的 drone 營造清晨霧氣繚繞的氛圍，使用三和弦的弦樂與鐘聲為專輯開場。程式碼中 `setcpm(100/4)` 使節奏緩慢，`room(0.8)` 增加空間感。

## 2. City Pulse（城市脈動）
這首歌模仿都市的節奏，鼓組使用 `s("bd*4")` 和 `s("hh*8")` 表現都市脈動；貝斯採用 sawtooth 合成器營造厚度，和弦進行取自 I–V–vi–IV。旋律快速穿梭，如同人在車陣中穿梭。

## 3. Retro Future（復古未來）
以搖滾鼓點為基礎，搭配 square 波的低音和 saw 波的 riff，營造復古龐克的感覺；chorus 區加上寬廣 pad，使舊時代與未來音色結合。`every(32, x => x.fast(2))` 用於橋段提升能量。

## 4. Island Dreamers（島嶼夢遊）
運用三角波模擬木吉他，旋律遵循五聲音階，表達對島嶼自然景色的嚮往。`slow(2)` 延長每個音符，讓音符間有餘韻，附帶背景 drone 讓歌曲更平靜。

## 5. Midnight Lovers（午夜戀人）
以抒情小調鋼琴為核心，低音持續支撐；簡單的鼓點讓旋律更突出。使用 `every(32, x => x.room(0.9))` 在某些段落增加殘響，象徵回憶的重疊。

## 6. Starlit Overpass（星光天橋）
強烈的四四拍鼓和大量 hi-hat 襯托舞曲氛圍；伴隨 filter sweep 的合成器 pad 以及快速的琶音，構成夜晚天橋的霓虹閃爍。

## 7. Homegrown Techno（自製科技）
使用隨機旋律 (`rand.range().segment()`) 和 perlin 噪聲模擬自製電子樂器的即興感，向美秀集團的實驗精神致敬。高 bpm 的 techno 節拍，產生賽博台客的科技感。

## 8. Sunset Boulevard（落日大道）
中板節奏搭配溫暖的三角波和弦，像夕陽餘暉般柔和。旋律線中偶爾跳至高八度，營造視野開闊的感覺。

## 9. Heartstrings（心之弦）
溫柔的弦樂和深沉的低音，表達內心細膩的情感；使用簡單的鼓點，凸顯旋律的流動。`degradeBy(0.3)` 為旋律增添呼吸感。

## 10. Euphoria（狂喜之終章）
專輯的最高潮，融合舞曲鼓組與華麗的琶音，並加上 perlin 噪聲調制的 pad，使情緒層層堆疊。`every(16, x => x.fast(2))` 在副歌提升張力，最後以空靈的殘響收尾。
