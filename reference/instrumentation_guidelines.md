# 聲音設計指南

為了讓作品更貼近台灣樂團的風格，可參考以下設計建議：

## 1. 美秀集團（Amazing Show）風格
- 混合 **搖滾、龐克與民謠** 的節奏，節奏快速、鼓聲強烈。
- 在 Strudel 中可使用 `sound("sawtooth")` 或 `sound("square")` 模擬電吉他，再搭配 `lpf(600-900)` 取得粗糙質感。
- 加入自製樂器般的聲響，如使用 `rand` 產生隨機旋律或 `perlin` 調制的噪聲，模擬實驗精神。
- 舞台魅力來自高能量與厚重低頻，可使用快速的鼓 pattern（例如 `s("bd sd bd sd")`) 和強烈的貝斯線。

## 2. 告五人（Accusefive）風格
- 樂風多元，包括 **獨立搖滾、民謠搖滾、流行搖滾、合成器流行、流行舞曲、謠曲與電子音樂**【784024295954952†L129-L137】。
- 使用明亮的大調或浪漫的小調，加上雙主唱對應的旋律線；可分別用兩條 `note()` patterns 表示互動。
- 合成器音色可以選擇 `sound("sine")` 或 `sound("triangle")`，加 `room()` 產生空靈感；舞曲節奏則搭配 `sound("sawtooth")`、`delay()` 等效果。

## 3. 通用建議
- 適時使用 `room()`, `delay()`, `pan()` 增加空間感與立體感。
- `every()`, `sometimes()` 可以用來製造即興與人味，例如：`sometimes(x => x.degradeBy(0.2))` 有 50% 機率刪除事件，增加不完美感。
- 利用 `perlin.range()` 控制動態與濾波 cutoff，使聲音更有生命力。
